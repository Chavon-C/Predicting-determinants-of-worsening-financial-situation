# Predicting-determinants-of-worsening-financial-situation
This project analyses individual-level predictors of worsening financial situation using data from the 2022 US General Social Survey (GSS).

Using logistic regression models, this project:

1.Identifies key socio-economic predictors

2.Compares multiple model specifications

3.Evaluates model performance

The data used in this project can be found here: https://gss.norc.org/get-the-data/stata.html
